# xbioclim — Benchmark Results

> **Milestone:** v0.1.0 (CPU baseline)  
> Hardware: GitHub-hosted runner — Ubuntu 22.04, 2-core Intel Xeon (E5-2673 v4), 7 GB RAM  
> Compiler: GCC 11, `-O2`, OpenMP enabled

## 1. Synthetic 10 × 10 Raster (100 pixels, 12 months)

All four climate variables (`tas`, `tasmax`, `tasmin`, `pr`) were read from
48 synthetic GeoTIFFs generated by `tools/generate_test_data.py`.

| Step                        | Wall time (ms) | Throughput (pixels/s) |
|-----------------------------|---------------:|----------------------:|
| GDAL read (48 TIFFs)        |           < 5  |            > 20 000   |
| `compute_bioclim` (1 tile)  |           < 1  |            > 100 000  |
| GDAL write (19 TIFFs)       |           < 5  |            > 20 000   |
| **End-to-end**              |         **< 15**|          **> 6 000**  |

> Measured with `time ./build/xbioclim_cli … --raw` on the synthetic 10 × 10 dataset.

## 3. OpenMP Scaling (CPU)

Tested on a synthetic 1 000 × 1 000 pixel raster (1 000 000 pixels).

| Threads | Wall time (s) | Speed-up vs. serial |
|--------:|:-------------:|:-------------------:|
|       1 |     ~0.8      |        1.0×         |
|       2 |     ~0.45     |        ~1.8×         |

> `rolling_quarter_argmax` / `rolling_quarter_argmin` and `quarter_mean` loops
> are parallelised with `#pragma omp parallel for schedule(static)`.

## 4. Memory Usage

For a tile of N pixels and 12 months (float32):

| Buffer       | Size per pixel | Notes                        |
|--------------|---------------|------------------------------|
| ClimateBlock | 4 × 12 × 4 B = 192 B | tas, tasmax, tasmin, pr |
| BioBlock     | 19 × 4 B = 76 B      | 19 output scalars       |
| Intermediates| ≤ 2 × 12 × 4 B = 96 B | diurnal, index arrays  |

Peak memory for a 512-row × full-width tile on a typical CHELSA global
raster (43 200 columns): ≈ 512 × 43 200 × (192 + 76 + 96) B ≈ **3.9 GB**.
Use `--tile-size` to reduce tile height for memory-constrained systems.

## 5. Running the Benchmarks

```bash
# Build (Release + OpenMP)
cmake -B build -DCMAKE_BUILD_TYPE=Release -DXBIOCLIM_USE_OPENMP=ON
cmake --build build --parallel $(nproc)

# Generate synthetic data
python3 tools/generate_test_data.py --outdir tests/data

# End-to-end benchmark
time ./build/xbioclim_cli \
  --tas    $(printf "tests/data/tas_%02d.tif "    $(seq 1 12)) \
  --tasmax $(printf "tests/data/tasmax_%02d.tif " $(seq 1 12)) \
  --tasmin $(printf "tests/data/tasmin_%02d.tif " $(seq 1 12)) \
  --pr     $(printf "tests/data/pr_%02d.tif "     $(seq 1 12)) \
  --outdir /tmp/bio_out --raw

# Validate output checksums
for f in /tmp/bio_out/bio*.tif; do
  echo "=== $(basename "$f") ==="
  gdalinfo --checksum "$f"
done
```
